- Dữ liệu lưu cực kỳ khó hiểu - ý là cấu trúc và định dạng khó hiểu:
	+ Bình thường thì là file index.html, nhưng đây thì lại là folder index.html, trong folder thì lại có 1 file metadata, rất là khó hiểu. Khi đổi đuôi sang HTML thì xem được 1 chút, nhưng không có thông tin giá trị
	+ Sau khi mò một lúc thì đối với nuclues market thì chỉ có những file part.1 có dữ liệu (về mặt hàng, giá cả, địa chỉ ship, mua bán bằng coin nào)... => mấy file .meta, file không đuôi, hay các file có định dạng khác...thì hầu như không có dữ liệu hữu ích
	=> Chỉ có file part.1 thì có dữ liệu, nên đối với folder nuclues thì phải lọc toàn bộ file có chứa part.1 ra, còn các file khác có thể bỏ qua.
